学习代码见：https://github.com/caojx-git/spring-security-learn

- Spring Security 原理
- Spring Security OAuth 原理
- 基于session+cookie 实现 用户名+密码登录、手机号+验证码登录、QQ社交登录、微信社交登录
- 基于Token 实现 用户名+密码登录、手机号+验证码登录、QQ社交登录、微信社交登录
- JWT Token 使用
- RBAC模型实现权限管理系统
